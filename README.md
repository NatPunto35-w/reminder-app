# Reminder App - Android Nativa

App Android nativa in Kotlin per gestire impegni e ricevere notifiche.

## 🚀 Avvio Rapido

### 1. Apri in Android Studio
- File → Open → Seleziona questa cartella

### 2. Sincronizza Gradle
- Android Studio sincronizzerà automaticamente
- Se errori: Build → Clean Project → Rebuild Project

### 3. Collega il telefono
- Abilita Debug USB su S22 Ultra
- Collega via USB

### 4. Avvia l'app
- Run → Run 'app' (oppure Shift+F10)
- Seleziona il tuo S22 Ultra

## ✨ Funzionalità

- ✅ Aggiungi impegni con data e ora
- ✅ Visualizza lista impegni
- ✅ Elimina impegni
- ✅ Notifiche intelligenti
- ✅ Design moderno con Jetpack Compose

## 📋 Requisiti

- Android Studio Giraffe+
- JDK 11+
- Android 8.0+ (API 26)

## 🔧 Troubleshooting

**Errore Gradle?**
- Build → Clean Project
- File → Invalidate Caches → Restart

**Notifiche non funzionano?**
- Verifica permessi su Android 13+
- Vai in Impostazioni → App → Reminder → Notifiche

## 📱 Versione Minima

- Android 8.0 (API 26)
- Target: Android 14 (API 34)
